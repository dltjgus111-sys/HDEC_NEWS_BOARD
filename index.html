반영 방법
① 아래 [config.json 내보내기] 클릭 → 파일 다운로드
② GitHub 저장소 → config.json → 연필 아이콘 → 내용 붙여넣기 → Commit
③ Actions 탭 → Run workflow (기다리면 다음날 아침 자동 반영)
